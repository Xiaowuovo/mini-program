# 🌱 云端小筑 - 共享菜园微信小程序

> 智慧农业 · 共享经济 · 绿色生活

---

## 📖 项目简介

云端小筑是一个基于微信小程序的共享菜园管理平台，让城市居民能够租用菜地，体验种植乐趣，同时提供远程监控、代管理等增值服务。

### 🎯 项目定位
- **类型**：毕业设计项目
- **目标**：提供完整的菜地租赁和管理解决方案
- **特色**：远程监控 + 社区互动 + 智能提醒

## 🚀 快速开始

### 环境要求
- Python 3.8+
- MySQL 8.0+
- 微信开发者工具
- Node.js 14+（可选）

### 1. 启动MySQL
```bash
net start MySQL
```

### 2. 初始化数据库
```bash
cd backend
init_database.bat
```

### 3. 启动后端服务
```bash
cd backend
run.bat
```

### 4. 启动小程序
1. 打开微信开发者工具
2. 导入 `miniprogram` 目录
3. 配置"不校验合法域名"
4. 编译运行

**详细教程**：查看 [快速开始指南.md](./快速开始指南.md)
