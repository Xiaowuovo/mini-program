## 🎨 图片规格

### 轮播图
- 格式：JPG
- 尺寸：750×300 像素
- 大小：< 200KB/张
- 主题：菜园、绿色植物

### 功能图标
- 格式：PNG（透明背景）
- 尺寸：81×81 像素
- 大小：< 20KB/张
- 颜色：绿色 #4CAF50

### 空状态图
- 格式：PNG
- 尺寸：240×240 像素
- 风格：简洁友好
